üßπ Data Cleaning Steps (SQL)

  • Removed duplicate records
  • Handled NULL values in numeric and categorical fields
  • Standardized company and industry names
  • Corrected inconsistent date formats
  • Converted numerical fields to correct data types
  • Filtered out invalid rows (negative layoffs, invalid percentages)

üß™ Exploratory Data Analysis (EDA)

Performed using SQL queries:

  • Layoffs by industry
  • Layoffs by company
  • Layoffs by country & region
  • Month-wise and year-wise trend analysis
  • Percentage of workforce laid off across industries
  • Funding vs layoffs analysis (relationship between fundraising stage & layoffs)

3Ô∏è‚É£ Yearly layoffs trend
